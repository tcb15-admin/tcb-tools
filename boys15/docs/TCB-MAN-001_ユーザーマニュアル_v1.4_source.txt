画面上部のステップ名
1 活動設定 → 2 欠席・当番 → 3 割振り結果

STEP 1：活動設定
1-1 活動パターンを選ぶ（左から「両方練習」「片方試合」「両方試合」）
ボタン	使う場面
片方試合	チームが 1 試合のみ（試合組と練習組に分かれる）
両方試合	チームが 2 試合（A・B グループそれぞれ試合）
両方練習	全員で練習（全道具を分担）

1-2 両方練習＋練習場所が分かれる場合
「両方練習」を選ぶと、「練習場所が分かれる（2 グループで会場ごとに割り振る…）」チェックが表示されます。オンにすると室内＝左グループ（A）、屋外＝右グループ（B）として割り振ります。道具ごとの「会場」はマスタ設定で指定します（未選択の道具は、試合用の A/B 列の区分に従います）。

1-3 活動日・グループ名
活動日は日付を選択。グループ名は最大 12 文字で変更可能（片方・両方試合、および練習でグループ名が使える画面）。

1-4 前回の結果を見る
「前回」ボタンで、直近保存された割振り結果のモーダルを開けます。

STEP 2：欠席・当番
2-1 割振り対象外を選ぶ
欠席・その他理由でこの日の割振り対象外にするメンバーをタップ（赤色）。休部・コーチは常に対象外。

2-2 お茶当番を選ぶ
翌活動日分のお茶当番メンバーをタップ（橙色）。割振り対象外になります。

2-3 道具の一時除外
この日だけ割振りから外したい道具をタップ（赤・「除外」バッジ）。もう一度タップで解除。マスタ設定には影響しません。

2-4 前回ベースの割振り（任意）
「前回の割振りを出発点にする」にチェックを入れると、STEP1 の「前回」に保存されている内容を出発点に、この画面で選んだ欠席・お茶当番・グループを反映し、対象外になった担当の道具だけを自動で振り直します。活動パターン（片方／両方試合／練習・場所分離）が前回と違う場合は使えません。

2-5 グループ振り分け（片方・両方試合のみ）
全員が B グループからスタート。試合参加メンバーをタップして A グループに切替。
ボタン	動作
デフォルト保存	現在の振り分けを保存 → master.json を GitHub にアップロードで全端末同期
デフォルト読込	保存した振り分けを呼び出す

2-6 割振りを実行
「割振りを実行」で結果を計算します。この時点では過去担当回数（PAST）はまだ増えません（試行・LINE 下書き用）。

STEP 3：割振り結果
3-1 実施確定（PAST への反映）【重要】
「割振りを実行」だけでは PAST は更新されません。保護者へ連絡した実際の持ち帰り内容になったら「この内容で実施確定（PAST に反映）」を押してください。同じ活動日で担当を直した場合は、もう一度押すとその日のカウントを差し替えます。GitHub へマスタを同期する場合は、続けてマスタ画面の「保存」も実行してください。

3-2 担当の手直し・元に戻す・公平に再分配
道具別一覧でドロップダウンから担当を変更できます。「元に戻す」「先に進む」で変更履歴を辿れます。
「公平に再分配」は、表のチェック列で対象にした道具と、担当が未割当の道具をまとめて再割当します（固定担当の道具は対象外）。担当 0 件の人を優先します。

3-3 タブ
「道具別」「メンバー別」「LINE 展開用」を切り替えられます。

3-4 道具別一覧
複数道具担当の行は黄色背景＋「複数」バッジで強調。ドロップダウンで手動変更可。「写真」で説明・写真を表示。

3-5 LINE 展開用テキスト
コピーボタンでクリップボードへ。複数担当道具に注意マーク。末尾に複数担当者のまとめ。

3-6 印刷プレビュー・PDF
「印刷/PDF」でプレビューを開きます。
ボタン	動作
閉じる	割振り結果画面に戻る
印刷 / PDF 保存	ブラウザの印刷ダイアログ（PDF 保存可）
PDF で共有	Web Share API が使える端末でのみ表示。PDF を生成して共有シートへ（失敗時はダウンロード案内）

iPhone で PDF を LINE に送る手順（例）
「印刷/PDF」→「印刷 / PDF 保存」または「PDF で共有」→ 端末の手順に従い LINE を選択。

割振り履歴
ヘッダの「履歴」から過去の試行結果を確認できます（「割振りを実行」時に端末へ保存された内容。実施確定後の PAST とは別の場合があります）。
機能	操作
詳細確認	活動日をタップ → 道具別担当者を表示
直近 1 ヶ月（活動日）	「直近 1 ヶ月（活動日）：一覧・PDF」で、今日から過去 30 日以内の活動日を道具×日付形式で表示。横長 PDF や印刷も可能
選択削除	チェックで選択 →「選択削除」
全削除	「全削除」→ 確認バーの「削除する」
書き出し	「JSON で書出」で全履歴をダウンロード

マスタ設定
右上の「マスタ」から。変更後は「保存」で master.json をダウンロードし GitHub にアップロード。
メンバー一覧・道具一覧はハンドルをドラッグ＆ドロップで並び替え。
両方練習・場所分離時は道具に「会場」列（室内／屋外／未選択）が使われます。

よくある質問
質問	回答
過去回数はいつ増える？	「この内容で実施確定（PAST に反映）」を押したとき。試行の「割振りを実行」だけでは増えない
スマホでマスタが古い	GitHub に master.json をアップロードしてページを更新
iPhone で PDF を送りたい	印刷プレビューの「PDF で共有」または印刷ダイアログから PDF 化
道具を一時的に除外したい	STEP2 の「道具の一時除外」
室内・屋外の練習に分けたい	両方練習でチェックをオンにし、マスタで会場を設定
